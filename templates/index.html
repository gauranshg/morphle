Morphle Live Mode Scanner

This system simulates a live mode scanner with the following features:

  • Keyboard-based movement: Use arrow keys to update the target cell.
  • Mouse-click target: Click inside the slide to set the target point directly.
  • Real-time updates: Both the current position marker and the target marker update immediately after each key click or mouse click, and then continue updating at variable intervals based on activity.
  • Simulated machine movement: The system uses a simple time.sleep to simulate movement delays.
  • Speeding up over distance: When moving multiple blocks, the delay for each block is calculated using the formula: delay = 3 * sqrt(n) - 3 * sqrt(n - 1), where n is the block number. This means the first block takes 3 seconds, the second around 1.24 seconds, and so on.
  • Optimized path: The machine does not follow the exact path provided by user commands; it always finds the closest (minimal) path toward the final target block.
